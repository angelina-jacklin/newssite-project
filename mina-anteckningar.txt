Koppla mapp i github till datorn:

cd = change directory

mkdir = make directory 

git clone https://github.com/angelina-jacklin/newssite-project.git = klistra in adressen till github-mappen du vill klona




Lägg till, spara och skicka ändringar i fil till github:

git add nissesfil.txt = lägg till ändring eller fil

git commit -m "hehdå" = spara ändringarna och skriv ett meddelande om vad du gjort för ändringar

git push = skicka dina ändringar till github





Övrigt:

git remote -v = kolla github adress

ls = lista filer

skriv det du vill ha hjälp med (ex. ls) --help = ta reda på vad ett kommando betyder och hur det används

notepad önskat-namn-på-notepad-filen = skapa och namnge en ny fil i notepad

notepad mina-anteckningar & = använd & efter om du vill kunna använda terminalen samtidigt som t.ex notepad

echo "Nissesfil" > nissesfil.txt = skriv in text i fil

diff = används för att jämföra filer rad för rad
